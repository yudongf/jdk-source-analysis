1. Redis 中的 Set

Redis 中的集合对象编码可以是：

  1. intset

  2. hashtable

转换的条件是：

  1. 集合对象保存的所有元素都是整数值；

  2. 集合对象保存的元素个数不超过 512 个；（通过参数 set-max-intset-entries 来调整，默认是 512）
